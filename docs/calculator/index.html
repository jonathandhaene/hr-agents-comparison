Why these numbers
  • AOAI session cost = (in_tokens × $/1K + out_tokens × $/1K) / 1000 × calls_per_session.
  • Copilot Studio cost = (gen_answers + classic + tenant_graph + tools + flows) × credits × $0.01.
  • Optimisation: with the M365 Copilot license model, generative/classic answers and tenant-graph grounding are billed at zero for in-scope HR scenarios. Copilot Studio agent flows and custom connector calls remain billable.
  • Solution A (M365 Agents SDK) never accrues Copilot Studio credits — its conversation surface is the Agents SDK / Bot Service, not Copilot Studio.
  • Solution C (Foundry hosted agent) bills only AOAI tokens for sessions; no Copilot Studio credits even without the M365 Copilot license.
  • Solution D (Mixed) uses Copilot Studio for the conversation surface (B-style billing) plus AOAI tokens only on UC4 + UC5-summary that delegate to Foundry.
